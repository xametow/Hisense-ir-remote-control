# Hisense-ir-remote-control
Hisense ir remote control with esp32c3 super mini. It power from 5v port in conditioner and pair with apple homekit.

История создания:
Создание умного модуля для управления через apple homekit, в том числе через HomePod mini. В кондиционере есть вайфай модуль для подключения к их приложению ConnectLife.TRIR, однако оно не особо удобное в ключе использования в умном доме, а так же не привязывается в умный дом apple, что создает неудобство при использовании, приходится использовать разные приложения для разных девайсов. В моем кондиционере уже был установлен вайфай модуль, который питается от 5 вольт. Разобрав его, мне нужна была только фишка для питания. Скорее всего фишки есть во всех кондиционерах, поскольку даже кондиционеры без вайфай модуля предполагают его доустановку самостоятельно. После получения питания начал разбираться как можно им управлять. В голову сразу пришла идея - ик порт. Собрав на коленке esp32 + ir transmittor и зареверсив пульт, все получилось. Однако дальность такого устройства при условии небольшого питания крайне мала. То есть нельзя по аналогии как с яндекс станцией поставить устройство на стол и использовать его для всех устройств с ик портом в доме. Нужно было повышать питание. МНе это не нужно, нужно было только управлять кондиционером. На расстоянии до 30см все работает отлично. В общем то, большего и не надо. Собрав и установив устройство в штатное крепление, направив ик сенсор в лед панель кондиционера все заработает. Начал собирать на esp32 , однако его энергопотребление в пике при подключении к вайфай может доходить до 500 мА. Выход 5в под штатный вайфай модуль выдает меньше (не замерял). Так что пришлось использовать esp32c3 super mini. На нем все заработало, но не сразу. Из-за одного ядра в плате, пакеты ик отправлялись раздельно и некорректно, из-за совместной рабботы с вайфай. Так что про готовые библиотеки пришлось забыть. В паре с Claude AI смог разработать скрипт, который отлично себя показал в деле.

Использование:
В папке src лежит скрипт .ino с дампами .h, после загрузки достаточно через консоль в Arduino IDE ввести свою файфай точку и устройство будет готово для подключения в apple homekit. В home нажать (+) -> добавить аксессуар -> Другие параметры... -> найти устройство и ввести код-пароль: 46637726. Далее устройство появится в приложении.
