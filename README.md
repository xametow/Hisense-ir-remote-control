# Hisense-ir-remote-control
# Hisense AC → Apple HomeKit via ESP32-C3

Модуль для управления кондиционером Hisense через Apple Home и Siri без стороннего приложения. Устанавливается вместо штатного Wi-Fi модуля и питается от него же.

![esp32c3 + IR transmitter установлены в кондиционер]

---

## Как это работает

Кондиционер управляется по ИК-порту. ESP32-C3 Super Mini подключается к разъёму питания штатного Wi-Fi модуля, направляем ИК-передатчик на приёмник (на светодиодную панель отображения температуры) кондиционера и эмулируем оригинальный пульт. Работает на расстоянии до 30 см из-за питания, но этого достаточно, если модуль стоит прямо внутри.

Устройство поднимает HomeKit-аксессуар в локальной сети. Управление через приложение Home, Siri и автоматизации работает без внешних серверов.

---

## Почему ESP32-C3, а не обычный ESP32

Штатный разъём питания в кондиционере рассчитан на небольшой Wi-Fi чип. Обычный ESP32 в пике потребляет до 500 мА — этого хватает, чтобы просесть по питанию и уйти в бесконечный ребут. Пробовал добавить конденсатор, но все безрезультатно. ESP32-C3 Super Mini значительно экономичнее и стабильно запускается от этого же разъёма.

---

## Почему без стандартных ИК-библиотек

ESP32-C3 - одноядерный чип. При активном Wi-Fi стек прерываний мешает программной генерации ИК-сигнала: пакеты уходят с искажёнными таймингами и кондиционер их не понимает. Стандартные библиотеки (IRremoteESP8266 и др.) используют именно программный bit-banging.

Решение - аппаратный блок RMT, который есть на ESP32-C3. Он генерирует ИК-импульсы на уровне железа, независимо от загрузки процессора. Вместо реконструкции протокола Kelon168 все коды сняты с оригинального пульта и хранятся как raw-дампы таймингов.

## Подключение

| Кондиционер | ESP32-C3 | IR Transmitter |
|---|---|---|
| 5V | VIN | VCC |
| GND | GND | GND |
| — | D4 | DAT |

---

## Установка

1. Открыть папку `src` в Arduino IDE
2. Установить библиотеку **HomeSpan** через Library Manager
3. Board: `ESP32C3 Dev Module`, USB CDC On Boot: `Enabled`
4. Залить прошивку
5. В Serial Monitor (115200) ввести `W` и следовать инструкции для подключения к Wi-Fi

**Добавление в Apple Home:**  
`+` → Добавить аксессуар → Другие параметры... → найти устройство → код `466-37-726`

## Управление голосом через Siri

- "Привет, Siri! Включи кондиционер."
- "Привет, Siri! Выключи кондиционер."
- "Привет, Siri! Кондиционер 20 градусов."
- "Привет, Siri! Максимальная скорость кондиционера."
- "Привет, Siri! Минимальная скорость кондиционера."
- "Привет, Siri! Скорость кондиционера 70." (0-100) где 0 - минимальная скорость, 100 - максимальная скорость.

## Структура репозитория

```
src/
  ac_homekit_rmt.ino      # основной скетч
  kelon_temp_fan_data.h   # raw ИК-коды (16–30°C × скорость 1–5)
dumps/
  raw_dumps.txt           # исходные дампы с оригинального пульта Hisense
                          # формат: 16/5 = 16°C, скорость вентилятора 5
3d_model/
  ...                     # модели корпуса для печати (крышки нет, замотал скотчем)
                          # для крепления IR Transmitter'а используется винт M3*5
```

---

## Поддерживаемые параметры

- Вкл / Выкл
- Температура: 16–30°C
- Скорость вентилятора: 5 уровней
